Overview

DMS (Deep Metabolic Simulation) is a good tool to simulate the modified metabolic network according to what users submit. To build a more robust, less data-dependent and more detailed Metabolism Simulation, our team build four models to deal with the reactions which are documented differently.

Users need to input the changes in the metabolic network and then our software will generate a list of the top 100 compounds whose content in the cell has changed greatly and a heat map to describe the changes of compound content.
